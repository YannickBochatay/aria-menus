Menu Web Components

Full accessible menus, context menus, menu bars and menu buttons, according to the ARIA Authoring Practices Guide.

Installation

CDN

<script> src="https://cdn.jsdelivr.net/npm/menu-wc" type="module" ></script>

NPM

npm install menu-wc
